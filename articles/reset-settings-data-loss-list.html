スマホ / 設定

動作が重いからって設定を適当にリセットすると消えるデータ一覧

スマホが重い時に、設定から「リセット」を押すと種類によって消えるものが全然違います。
「設定だけ戻る」つもりで触ったのに、Wi‑Fiが全部消えた／壁紙や通知が戻った／最悪、初期化して写真も消えた…がありがち。
ここでは“どのリセットで何が消えるか”を、iPhone/Android別に一覧でまとめます。

  • 結論：リセットの種類を間違えると痛い
  • iPhone：リセット別「消えるもの/消えないもの」
  • Android：リセット別「消えるもの/消えないもの」
  • 重い時に“安全に”試す順番
  • やる前に確認するバックアップ
  • よくある質問

大事：「リセット」＝「初期化」じゃないこともあるけど、初期化に近い操作も混ざっています。
まずは名称を見て選ぶのがコツ。

結論：リセットの種類を間違えると痛い

設定リセット（軽め）
  • 通知/表示/プライバシーなどの設定が初期状態に戻る
  • 写真やトーク履歴など、データ本体は消えないことが多い
ネットワークリセット（地味に困る）
  • Wi‑Fiパスワード、Bluetooth、APNなどが消える
  • 「学校Wi‑Fi」「家Wi‑Fi」入れ直しになる
初期化（工場出荷状態）＝データ消去
  • 端末がまっさらになる（アプリ・写真・アカウント等）
  • バックアップがないと戻らない

iPhone：リセット別「消えるもの / 消えないもの」

「すべての設定をリセット」（軽め）

端末の設定だけを初期状態に戻す系。データ消去ではありません。

  • 消える/戻ることがある：壁紙、通知設定、位置情報の許可、プライバシー設定、キーボード設定、画面表示、アクセシビリティ設定 など
  • 基本消えない：写真/動画、連絡先、メモ、アプリ本体、LINEのトーク（端末内）※

※アプリによって「通知許可」や「権限」が戻ることがあるので、使いづらくなることはあります。

「ネットワーク設定をリセット」（Wi‑Fi周りが消える）
  • 消える：保存したWi‑Fi（SSID/パスワード）、VPN設定、モバイル通信設定の一部、Bluetoothのペアリング など
  • 消えない：写真/アプリ/データ

「Wi‑Fi繋がらない」時は有効だけど、学校や家のWi‑Fiを入れ直しになります。

「キーボードの変換学習をリセット」（地味に戻る）
  • 消える：予測変換の学習、よく使う単語の学習
  • 消えない：他のデータ全般
「すべてのコンテンツと設定を消去」（初期化）

これは端末のデータを消して工場出荷状態にする操作。重いからって気軽にやると危険です。

  • 消える：アプリ、写真/動画（端末にしかない分）、端末内データ、設定、アカウント情報 など
  • 残る可能性：iCloudに同期されている写真/連絡先/メモ等（サインイン後に復元される）

「iCloudに同期されてるから大丈夫」と思っても、同期が止まっていたなら消えます。必ずバックアップ確認（後述）。

Android：リセット別「消えるもの / 消えないもの」

Androidは機種（Pixel/Galaxy/Xperiaなど）で名称が少し違うけど、だいたい次の3系統です。

「設定のリセット（アプリ設定をリセット）」系
  • 消える/戻ることがある：既定のアプリ、通知の許可、権限、バックグラウンド制限、Wi‑Fiの自動接続設定の一部 など
  • 基本消えない：写真/動画、アプリのデータ本体、アカウント
「ネットワークのリセット」
  • 消える：保存Wi‑Fi、Bluetoothペアリング、モバイル通信設定の一部（APNなど）
  • 消えない：写真/アプリ/データ
「工場出荷状態にリセット（初期化）」
  • 消える：端末内の写真/動画、アプリ、アプリ内データ、ダウンロード、設定、ログイン情報 など
  • 残る可能性：Googleに同期されてる連絡先/カレンダー、Googleフォトにバックアップ済みの写真 など

これも「バックアップしてる前提」の操作。バックアップ設定がOFFだと、終わります。

重い時に“安全に”試す順番（おすすめ）

リセット前にまずやる（安全で効果が出やすい）
  1. 再起動（意外と効く）
  2. ストレージ空き容量を作る（写真整理/不要アプリ削除）
  3. OS/アプリ更新（不具合修正が入る）
  4. Safari/Chromeのキャッシュ削除（ブラウザが重い場合）
リセットするなら軽い順に
  1. （必要なら）ネットワーク設定のリセット
  2. すべての設定をリセット（iPhone）/アプリ設定のリセット（Android）
  3. 最終手段：初期化（工場出荷状態）※バックアップ必須

“重い＝初期化”は飛びすぎ。まずは空き容量と再起動が正攻法です。

やる前に確認するバックアップ（これだけは見て）

iPhone

  • 設定 → 自分の名前 → iCloud → iCloudバックアップ
  • 写真：iCloud写真がONか、同期が止まってないか
  • LINE：トーク履歴のバックアップ（iCloud/Google Drive）

※「容量不足」で止まってる人が多いので注意。

Android

  • 設定 → システム → バックアップ（機種で名称違い）
  • 写真：GoogleフォトのバックアップがONか
  • LINE：トーク履歴バックアップ（Google Drive）

よくある質問

Q. 「すべての設定をリセット」って写真は消える？

基本は消えません（設定が初期状態に戻るだけ）。ただし通知や権限などが戻って「使いにくい」ことはあります。

Q. ネットワークリセットすると何が困る？

Wi‑FiパスワードやBluetoothが消えるので、家/学校のWi‑Fiを入れ直し、イヤホン等を再ペアリングする必要が出ます。

Q. 初期化したら、クラウドにある写真も消える？

端末だけ消すならクラウド側は残ることが多いですが、設定や同期状況によって違います。 「同期が止まってた」分は消えるので、初期化前にバックアップ確認が必須です。
