安全 / ネット

「ウイルスに感染しました」って画面に出た。それ、本物じゃなくて広告かも

こういう「ウイルスに感染しました」「今すぐスキャン」「警告！」みたいな画面、ほとんどは本物のウイルス警告じゃなくて“偽の広告（脅し）”です。
目的は、アプリを入れさせたり、電話させたり、個人情報やカード情報を取ること。
この記事では見分け方 → その場の消し方 → 再発防止まで、端末別にまとめます。

  • まずこれ：絶対に押さない
  • 本物っぽく見える理由と見分け方
  • iPhone（Safari）で消す
  • Android（Chrome）で消す
  • PC（Windows/Mac）で消す
  • 再発防止：通知・拡張機能・アプリをチェック
  • もし押してしまった/入れてしまった
  • よくある質問

超重要：画面内の「OK」「閉じる」「スキャン」ボタンは押さない。
消すときはブラウザ自体を閉じる／履歴・サイトデータを消すのが安全。

まずこれ：絶対に押さない

やってはいけない
  • 画面のOK / Cancel / 閉じる / スキャン開始を押す
  • 表示された電話番号にかける
  • 「ウイルス除去」アプリをその場で入れる
  • Apple ID/Google/クレカ情報を入力する
安全な行動
  • ブラウザのタブを閉じる（無理ならアプリごと終了）
  • 履歴・サイトデータを削除（後で手順あり）
  • 同じサイトの通知許可を外す

本物っぽく見える理由と見分け方

見分けの基本
  • ブラウザ内で出てる（Safari/Chromeのページっぽい）→ 偽の可能性が高い
  • 電話番号が書いてある → ほぼ詐欺
  • カウントダウンや警告音で焦らせる → 典型的
  • 「今すぐ〇〇を入れて」など行動を強制 → 典型的

本物のセキュリティ警告は、公式の設定画面やセキュリティソフト内で出ることが多く、 “いきなりWebページで電話させる”みたいな流れはかなり怪しいです。

iPhone（Safari）で消す

1) まずページを閉じる（押さずに）
  1. Safariのタブ一覧を開いて、そのタブを閉じる
  2. 閉じられない場合はホームに戻る → アプリスイッチャー → Safariを終了
2) 履歴/サイトデータを削除
  1. 設定 → Safari
  2. 履歴とWebサイトデータを消去

同じ広告が出続ける時は、サイト側のデータ（キャッシュ/クッキー）を消すのが効きます。

3) 再発防止の設定

  • 設定 → Safari → ポップアップをブロック をオン
  • 設定 → Safari → 詐欺Webサイトの警告 をオン

4) 変なものが入ってないか確認

  • 設定 → 一般 → VPNとデバイス管理（見知らぬプロファイルがあれば注意）
  • カレンダーに不審な予定が増えたら「購読カレンダー」も確認

iPhoneで「アプリ入れてスキャンしろ」はほぼ罠。App Store以外に誘導されたら特に危険です。

Android（Chrome）で消す

1) タブを閉じる（無理ならChromeを強制終了）

タブが閉じられないなら、設定→アプリ→Chrome→強制停止で止めてOKです。

2) Chromeの閲覧データ削除
  1. Chrome → ︙ → 履歴 → 閲覧履歴データを削除
  2. 「Cookie とサイトデータ」「キャッシュされた画像とファイル」を選ぶ

3) 通知を許可してないか確認

「許可」を押してしまうと、怪しい通知が飛び続けます。
Chrome → 設定 → サイトの設定 → 通知 で、見知らぬサイトをブロック/削除。

4) 最近入れたアプリを見直す

直前に入れたアプリが原因のこともあります。怪しいアプリはアンインストール。 可能ならGoogle PlayのPlay Protectもオンに。

PC（Windows/Mac）で消す

1) 画面が閉じない時は“ブラウザごと”終了
  • Windows：Ctrl + Shift + Esc → タスクマネージャーでブラウザを終了
  • Mac：Command + Option + Esc → ブラウザを強制終了

“ページの×ボタン”を押すより、ブラウザを閉じた方が安全です（偽ボタンの可能性があるため）。

2) 拡張機能（アドオン）を確認

変な拡張機能が入ると広告が出続けます。怪しいものは削除。

3) ブラウザのリセットも有効

Chrome/Edgeには設定を初期化する機能があります。直らない時の最終手段として使えます。

再発防止：通知・拡張機能・アプリをチェック

サイト通知は基本「許可しない」

「通知を許可しますか？」は、急いでる時ほど押しがち。
迷ったらブロックが安全です。

“無料クリーナー/高速化”系は注意

不要な広告アプリや、紛らわしい最適化アプリが原因のこともあります。

OSとブラウザを更新

古い環境ほど悪用されやすい。アップデートは地味に効きます。

もし押してしまった/入れてしまった

被害を広げないための手順
  1. そのページ/アプリを閉じる（PCならブラウザ強制終了）
  2. 入れてしまったアプリがあればアンインストール
  3. 怪しいサイトにログインしたなら、パスワード変更＋二段階認証
  4. カード情報を入れた/電話した場合は、家族やカード会社に相談

「スキャンが必要」と言われても焦らない。まずは押さずに閉じるが正解です。

よくある質問

Q. これって“開いただけ”で感染する？

多くは脅し広告です。開いただけで即アウトというより、ボタンを押す/アプリを入れる/情報を入力することで被害になります。 ただしゼロではないので、早めに閉じてデータ削除が安全です。

Q. 「閉じる」を押すのもダメ？

画面内の「閉じる」は偽物の場合があります。できるだけタブを閉じる/ブラウザを終了で対処するのが安全です。

Q. 何回も同じ警告が出る…

履歴・サイトデータが残っている、通知許可してしまった、怪しい拡張機能/アプリが入った、のどれかが多いです。 このページの手順を上から順にやれば、だいたい止まります。
