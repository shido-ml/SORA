Xでアカウントを一時停止・解除する方法

Xの「一時停止（無効化）」はすぐ消えるように見えますが、実は30日間の猶予があります。
その間にログインすれば解除（再開）できます。やり方と注意点を、迷わない順でまとめます。

一時停止の前に知っておくこと（重要）

30日ルール

一時停止するとプロフィールは見えなくなりますが、30日以内にログインすれば復活できます。
30日を過ぎるとアカウントは削除され、基本的に戻せません。

消えるのは“表示”

一時停止中は、あなたのプロフィールやポストは見えなくなります。
ただし検索エンジンの表示など、外部に残ることがあります（X側では操作できません）。

データを残したいなら

必要なら、一時停止の前にデータのダウンロードを検討。
ログインできない/パスワード不明だと、解除もできなくなりがちです。

注意 「凍結（suspended）」とは別

ここでの「一時停止」はあなたが自分で行う無効化です。
ルール違反などでアカウントが凍結/ロックされている場合は、解除方法が別になります。

iPhone/iPadアプリ：一時停止する手順

  1. Xアプリを開き、下のプロフィールアイコン → 設定とプライバシー
  2. アカウント（または「アカウント情報」）を開く
  3. アカウントを停止（無効化） を選ぶ
  4. 注意事項を読み、下の停止（無効化）
  5. 確認のためにパスワード入力 → 完了

アプリの表示名は更新で少し変わることがあります。
見つからない場合は、次の「PCブラウザ手順」でやるのが確実です。

Androidアプリ：一時停止する手順

  1. Xアプリ → プロフィールアイコン → 設定とプライバシー
  2. アカウント を開く
  3. アカウントを停止（無効化）
  4. 確認 → 停止（無効化） → パスワード入力

※Androidも、見つからない時はPCブラウザ経由が確実です。

PCブラウザ：一時停止する手順（いちばん確実）

  1. x.com を開いてログイン
  2. 左メニューの もっと見る → 設定とプライバシー
  3. アカウント → アカウントを停止（無効化）
  4. 注意事項を確認 → 停止（無効化）
  5. パスワード入力 → 完了
コツ 止める前に確認するもの
  • 登録メール/電話番号が今も使えるか
  • パスワードが合っているか（不安なら先に変更）
  • 二段階認証を使っている場合、認証アプリ/コードを用意

解除（再開）する方法

一時停止を解除するのは簡単で、基本はログインするだけです。

  1. Xアプリ または x.com を開く
  2. ユーザー名 / メール / 電話番号とパスワードでログイン
  3. 画面の案内に従い、再開（reactivate）を確定

復活後、フォロー/表示が整うまで少し時間がかかることがあります。
まずは数分待ってから再確認するとスムーズです。

うまく解除できない時の対処

パスワードが分からない

ログイン画面の「パスワードをお忘れですか？」から再設定。

メール/電話が使えない

登録情報にアクセスできないと復活が難しいです。可能なら、以前の端末/メールで確認。

30日を過ぎたかも

30日経過後は基本的に復活できません。ユーザー名/メールが再利用可能になる場合があります。

注意 「凍結」や「ロック」は別ルート

ログイン時に「凍結」「制限」などが出る場合は、一時停止の解除ではなく異議申し立て/認証が必要な可能性があります。

よくある質問

Q. 一時停止したら相手にはどう見える？

プロフィールが開けなくなり、基本的に表示されません。過去の言及（@）は文字として残ることがあります。

Q. 解除したら全部戻る？

30日以内なら戻ることが多いです。ただし反映に時間がかかる場合があります。

Q. 完全に消したい時は？

一時停止して30日ログインしないと削除に進む仕組みです。迷っているなら、まず一時停止が安全です。
