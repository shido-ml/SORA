【2026年版】インスタDMを既読つけずに読む方法｜既読（Seen）をオフにする設定と注意点

結論：今でも可能。Instagramの「既読表示（Read receipts）」をオフにすれば、DMを開いても相手に“既読/Seen”が出にくい。全体/特定チャット別の設定と注意点をまとめました。

公開日：2026-01-31 目安：3〜6分 対象：iPhone / Android

結論：今でもできます。いちばん安全で確実なのは、Instagramの「既読表示（Read receipts）」をオフにする方法。さらに「通知のプレビュー」で内容を読む方法も紹介します。

まず確認：インスタの“既読”はオフにできる

最近のInstagramでは、DMの既読表示（Seen / Read receipts）をオフにできるようになっています。設定を変えれば、DMを開いても相手側に「既読」が出にくくなります。

おすすめはこの順番

  1. 設定で「既読表示」をオフ（公式） ← いちばん確実
  2. 通知のプレビューで内容だけ見る ← ちょい確認向き

方法①（おすすめ）：既読表示（Read receipts）をオフにする

全体（すべてのDM）でオフにする手順

  1. インスタを開く → プロフィール（右下）
  2. 右上の≡（メニュー） → 設定とプライバシー
  3. 「メッセージとストーリーズへの返信」を開く
  4. 「既読表示（Show read receipts）」をOFF

※メニュー名はアップデートで微妙に変わることがあります。「read receipts / 既読表示」で探すと早いです。

特定の相手（そのチャットだけ）オフにする手順

  1. DM一覧から対象のチャットを開く
  2. 画面上の名前（相手名/グループ名）をタップ
  3. 「プライバシーと安全」を開く
  4. 「既読表示（Read receipts）」をOFF

注意：オフにすると「自分も相手の既読」が見えないことが多い

“自分だけ得する”設定ではなく、相手の既読も表示されなくなるパターンがあります。困る人は「特定チャットだけOFF」がおすすめ。

方法②：通知プレビューで“内容だけ”読む（短文向け）

「開かずに内容だけ見たい」なら、通知のプレビューで読むのが安全です（ただし長文は切れることがあります）。

iPhone

  • 設定 → 通知 → Instagram
  • プレビュー表示（ロック画面/常に など）を調整
  • ロック画面や通知センターで内容を確認

Android

  • 設定 → 通知 → アプリ → Instagram
  • 通知の内容表示（ロック画面の表示）を調整
  • 通知から内容を確認

この方法は「DMを開かない」ので既読が付く心配が少ないです。まずは通知で確認→返信するなら開くが鉄板。

よくある疑問

Q. 既読表示をオフにしたら、相手に通知される？

基本的に「既読表示をオフにしたこと」が通知でバレるケースは少ないです。ただし相手が「既読が付かない」ことで気づく可能性はあります。

Q. グループDMでも使える？

アプリのバージョンやチャットの種類で挙動が変わることがあります。まずは「特定チャットだけOFF」を試すのが安全です。

Q. 既読が付いた…取り消せる？

一度付いた既読を「なかったこと」に戻すのは基本できません。次からは先に既読表示をOFFにしておくのが確実。

まとめ

  • 今でも「既読をつけずに読む」ことは可能
  • 一番確実：既読表示（Read receipts）をOFF
  • ちょい確認：通知プレビューで内容だけ読む
